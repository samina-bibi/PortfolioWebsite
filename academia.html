During this current academic year, I have been taking part in a vertically integrated project (VIP). This is a multidisciplinary research project. During the first semester, I looked into comparing two storage systems that can be used by one of the programs used in the project.

This proccess involved me using Java and MySQL to compare the times of using an OutputStream (used in Java to write sounds being recorded) and using a MySQL database for the same purpose.
